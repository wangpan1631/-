* 常见的**时间复杂度(Big O notation)**
- O(1): Constant Complexity常数复杂度
- O(log n): Logarithmic Complexity对数复杂度
- O(n): Linear Complexity线性时间复杂度
- O(n^2): N square Complexity平方
- O(n^3): N cubic Complexity立方
- O(2^n): Exponential Growth指数
- O(n!): Factorial阶乘

* **空间复杂度**
- 数组的长度就是空间复杂度
- 递归的深度（特殊说明）是空间复杂度
