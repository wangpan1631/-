### MySQL服务的启动、停止与卸载，在 Windows 命令提示符下运行:
- 启动: net start MySQL
- 停止: net stop MySQL
- 卸载: sc delete MySQL
- 命令启动不了的话，需要通过计算机管理 --> 服务和应用程序 --> 服务 --> 找到mysql ，点击启动的方式来启动(windows)
